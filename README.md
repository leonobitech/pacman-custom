# Pac-Man Custom Edition

A custom Pac-Man game built with vanilla HTML5 Canvas and JavaScript.

## Features

- **Custom Characters**: OpenAI logo as Pac-Man, pixel crab sprites as ghosts
- **Claude Power Pellets**: Collect Claude logos to activate fright mode
- **Theme Switcher**: Toggle between Classic (blue) and Claude (terracotta/cream) palettes
- **Ghost AI**: Classic Pac-Man ghost behaviors (Blinky, Pinky, Inky, Clyde)
- **Responsive**: Fits any screen size, mobile touch controls included
- **Pause**: Press `P` or `Escape` to pause/resume

## Controls

| Key | Action |
|-----|--------|
| Arrow Keys / WASD | Move |
| Enter / Space | Start game |
| P / Escape | Pause |

## Run

Open `index.html` in your browser. No build step needed.

## Tech

- HTML5 Canvas
- Vanilla JavaScript
- Zero dependencies
